# CV 📄
Projeto desenvolvido para ser um Curriculum Vitae (CV) online. O projeto é um CV online para facilitar a interação do RH.

#### Funcionalidades
* CV online
* Download do arquivo (CV) em .pdf
* App responsivo
* Light/Dark mode
* Fácil interação
